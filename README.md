# Game Of Life

[Conway's Game of Life](https://en.wikipedia.org/wiki/Conway%27s_Game_of_Life)

## Building and Run

```
$ git clone git@github.com:aitorfernandez/game-of-life.git
$ cd game-of-life
$ yarn
$ yarn start
$ open index.html
```
